# EleProtect v2.0

EleProtect is a computational genomics tool for analyzing TP53 hotspot conservation across species with exploratory machine learning ranking.

## Features

- TP53 hotspot mapping
- DNA or protein input support
- Exploratory ML scoring
- CSV export
- Deployable Streamlit interface

## Deployment

### Local Run
```bash
pip install -r requirements.txt
streamlit run app.py
